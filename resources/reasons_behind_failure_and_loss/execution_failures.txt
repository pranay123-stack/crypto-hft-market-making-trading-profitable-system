================================================================================
                        EXECUTION FAILURES LOG
================================================================================
               Order Execution and Trade Failures
================================================================================

PURPOSE:
--------
This log tracks all order execution failures, rejections, and anomalies that
prevent successful order placement, routing, and fulfillment. In HFT, execution
quality is critical - even minor execution issues can eliminate profitability.

SCOPE:
------
- Order rejections by exchange
- Order routing failures
- Partial fill issues
- Price slippage problems
- Order state inconsistencies
- FIX protocol errors
- Gateway failures
- Order management system errors
- Position limit violations
- Credit/risk check failures
