================================================================================
                        DATA QUALITY ISSUES LOG
================================================================================
              Market Data and Reference Data Quality Problems
================================================================================

PURPOSE:
--------
This log tracks all data quality issues that can affect trading decisions,
including corrupt market data, missing prices, incorrect reference data,
timestamp issues, and sequence number gaps. Bad data can lead to incorrect
trading signals and significant losses.

SCOPE:
------
- Corrupt/malformed market data
- Missing price updates
- Stale data detection
- Sequence number gaps
- Timestamp anomalies
- Reference data errors
- Corporate actions data issues
- Symbol mapping errors
- Bid-ask spread anomalies
- Volume data errors
- Trade cancellations/busts
