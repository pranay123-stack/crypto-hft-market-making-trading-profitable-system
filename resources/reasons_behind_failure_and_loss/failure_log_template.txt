================================================================================
                          FAILURE LOG TEMPLATE
================================================================================
                    QUICK INCIDENT CAPTURE FORMAT
================================================================================

PURPOSE:
--------
This template provides a standardized format for immediate incident logging.
Use this for quick capture during or immediately after an incident occurs.
More detailed analysis should follow using category-specific templates.

INSTRUCTIONS:
-------------
1. Fill out this form as soon as an incident is detected
2. Complete all mandatory fields (marked with *)
3. Be factual and precise - avoid speculation in initial report
4. Update the log as new information becomes available
5. Reference this incident ID in all related documentation
