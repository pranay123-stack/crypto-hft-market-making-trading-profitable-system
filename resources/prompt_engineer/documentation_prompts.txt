================================================================================
AI PROMPTS FOR DOCUMENTATION GENERATION
================================================================================

These prompts help generate comprehensive documentation for HFT systems.
Use with Claude, GPT-4, or other LLMs to create clear, useful documentation.

⚠️ Always review and validate AI-generated documentation for accuracy!

================================================================================

================================================================================


================================================================================
DOCUMENTATION BEST PRACTICES
================================================================================

**1. Clarity**
- Write for your audience
- Use simple language
- Define technical terms
- Provide examples

**2. Structure**
- Logical organization
- Clear headings
- Table of contents
- Cross-references

**3. Completeness**
- Cover all features
- Include edge cases
- Document limitations
- Provide troubleshooting

**4. Accuracy**
- Keep docs up-to-date
- Review after code changes
- Test examples
- Version documentation

**5. Discoverability**
- Good search keywords
- Consistent naming
- Index/glossary
- Links to related docs

**6. Visual Aids**
- Diagrams for architecture
- Flowcharts for processes
- Screenshots for UIs
- Code examples

**7. Maintenance**
- Regular reviews
- Deprecation notices
- Version history
- Ownership
