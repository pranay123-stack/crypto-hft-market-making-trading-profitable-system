                    1. P&L & COST RELATIONSHIP OVERVIEW
================================================================================

FUNDAMENTAL RELATIONSHIP:
-------------------------

Net P&L = Gross P&L - Total Costs

Where:
    Gross P&L = Trading profits before any costs
    Total Costs = All operational costs (infra, trading, data, cloud)
    Net P&L = Actual profit after all costs

For HFT, this relationship is CRITICAL because:
    - Margins are razor-thin (often <0.1% per trade)
    - Costs can easily exceed profits if not managed
    - A 10% cost reduction = significant P&L improvement

WHAT IS BREAK-EVEN?
-------------------

Break-Even = Point where Net P&L = 0
At break-even: Gross P&L = Total Costs

Below break-even: Losing money
Above break-even: Making profit

    Trading costs: ~$2.20 per $1000 traded
    Data (usage-based): ~$0.10 per $1000 traded

                    5. COST-ADJUSTED PERFORMANCE METRICS
================================================================================

TRADITIONAL VS COST-ADJUSTED METRICS:
--------------------------------------

Traditional metrics ignore costs:
    - Gross Return
    - Gross Sharpe Ratio
    - Gross Win Rate

Cost-adjusted metrics include ALL costs:
    - Net Return (after all costs)
    - Cost-Adjusted Sharpe Ratio
    - Cost-Adjusted Alpha
    - Cost-Adjusted Information Ratio

COST-ADJUSTED SHARPE RATIO:
---------------------------

Traditional Sharpe Ratio:
    Sharpe = Mean(Gross Returns) / StdDev(Gross Returns)

Cost-Adjusted Sharpe Ratio:
    Cost-Adjusted Sharpe = Mean(Net Returns) / StdDev(Net Returns)

Example: MM_SPY_001
    Gross Daily Return: $84,250
    Daily Costs: $11,050
    Net Daily Return: $73,200

    Gross Daily Returns (60 days):
        Mean: $84,250
        StdDev: $22,140
        Gross Sharpe: $84,250 / $22,140 = 3.81

    Net Daily Returns (60 days):
        Mean: $73,200
        StdDev: $22,140 (same, costs don't add volatility)
        Cost-Adjusted Sharpe: $73,200 / $22,140 = 3.31

Impact of Costs: Sharpe reduced by 13.1%

For a high-cost, low-margin strategy:
    Gross Sharpe: 2.5
    Cost-Adjusted Sharpe: 1.2 (52% reduction!)

COST-ADJUSTED ALPHA:
--------------------

Alpha = Returns - (Risk-Free Rate + Beta Ã— Market Return)

Cost-Adjusted Alpha = Net Returns - (Risk-Free Rate + Beta Ã— Market Return)

Example:
    Gross Alpha: 8.5% annualized
    Costs: 3.2% of capital annualized
    Cost-Adjusted Alpha: 5.3% annualized

    Costs consumed 38% of alpha!

COST-ADJUSTED INFORMATION RATIO:
---------------------------------

Information Ratio = Alpha / Tracking Error

Cost-Adjusted IR = Cost-Adjusted Alpha / Tracking Error

Higher costs â†’ Lower alpha â†’ Lower Information Ratio

RETURN ON INVESTMENT (ROI):
---------------------------

ROI = Net P&L / Costs

For MM_SPY_001:
    Net P&L: $73,200/day
    Costs: $11,050/day
    ROI: 662% daily

    Annualized (252 trading days):
    Annual Net P&L: $18,446,400
    Annual Costs: $2,784,600
    Annual ROI: 662%

This means every dollar spent on costs generates $6.62 in net profit.

Excellent ROI!

ROI Benchmarks:
    Excellent: >500%
    Good: 200-500%
    Acceptable: 100-200%
    Poor: 50-100%
    Unprofitable: <50%

COST EFFICIENCY SCORE:
-----------------------

We developed a Cost Efficiency Score (0-100):

CES = 100 Ã— (1 - Cost_Ratio) Ã— (ROI / 10)

Components:
    - Cost Ratio (lower is better)
    - ROI (higher is better)

Example: MM_SPY_001
    Cost Ratio: 13.1%
    ROI: 662%
    CES = 100 Ã— (1 - 0.131) Ã— (6.62 / 10)
        = 100 Ã— 0.869 Ã— 0.662
        = 57.5

CES Interpretation:
    90-100: Exceptional
    70-90: Excellent
    50-70: Good
    30-50: Fair
    <30: Poor

Our strategy scores 57.5 (Good).
