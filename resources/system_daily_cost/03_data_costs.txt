                    3. CONSOLIDATED DATA FEEDS
================================================================================

UNDERSTANDING CONSOLIDATED FEEDS:
----------------------------------

Instead of subscribing to each exchange individually, consolidated feeds
aggregate data from multiple exchanges:

Advantages:
    - Single connection (lower infrastructure cost)
    - Normalized format (easier to process)
    - Cross-market analytics
    - Often cheaper than individual feeds

Disadvantages:
    - Slightly higher latency (1-5 milliseconds)
    - Less control over data quality
    - Vendor lock-in risk
    - May include unnecessary data

CONSOLIDATED TAPE SYSTEM (CTS/CQS):
------------------------------------

For U.S. equities, the Consolidated Tape provides:
    - CTS (Consolidated Tape System): Last sale prices
    - CQS (Consolidated Quote System): Best bid/offer from all exchanges

Pricing:
    Non-Display Professional Subscriber: $3,000/month = $99/day
    Data Capacity Fee: $2,500/month = $82/day
    Total: $181/day

Note: We still need direct exchange feeds for full depth-of-book,
      so this is supplemental, not a replacement.

WHEN TO USE CONSOLIDATED VS DIRECT:
------------------------------------

Use Direct Exchange Feeds When:
    ✓ Latency critical (<1 millisecond matters)
    ✓ Need full market depth (Level 2/3)
    ✓ Trading on specific exchange (DMA)
    ✓ Need exchange-specific order types

Use Consolidated Feeds When:
    ✓ Multi-market analysis
    ✓ Historical research
    ✓ Risk management (latency less critical)
    ✓ Backtesting and development
    ✓ International markets (direct feeds expensive)

Our Strategy:
    - Direct feeds for U.S. equities/futures (primary markets)
    - Consolidated feeds for international markets
    - Consolidated feeds for backup and development

This hybrid approach balances cost and performance.
